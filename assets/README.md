# 📁 Pasta Assets

Coloque os seguintes arquivos nesta pasta:

## 🖼️ capa_album.jpeg
- **Descrição**: Foto sua com sua amada (será exibida como capa do álbum)
- **Tamanho recomendado**: 280x280px ou superior
- **Formato**: JPEG, PNG ou qualquer formato de imagem
- **Localização**: `/assets/capa_album.jpeg`

## 🖼️ image2.jpeg
- **Descrição**: Foto do casal para o segundo story
- **Tamanho recomendado**: proporção quadrada ou vertical, com boa resolução
- **Formato**: JPEG, PNG ou qualquer formato de imagem
- **Localização**: `/assets/image2.jpeg`

## 🎵 musica_tema.mp3
- **Descrição**: Música "Gabriel Elias - Pequena Flor | Acústico"
- **Formato**: MP3 (ou qualquer formato suportado por HTML5 Audio)
- **Localização**: `/assets/musica_tema.mp3`

---

Os arquivos foram mencionados, mas você precisa adicioná-los manualmente a esta pasta para que o player funcione corretamente.
